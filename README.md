# MPI-реализация алгоритма Δ-шага для поиска кратчайших путей

## Описание проекта
Распределённая реализация алгоритма **Δ-шага (Delta-Stepping)** для решения задачи поиска кратчайших путей из одной вершины во взвешенных графах. Проект использует библиотеку MPI для параллельных вычислений на многопроцессорных системах.

## Алгоритмическая основа
Алгоритм Δ-шага — это параллельная версия алгоритма Дейкстры, которая:
- Разбивает рёбра на **лёгкие** (вес ≤ Δ) и **тяжёлые** (вес > Δ)
- Обрабатывает вершины **пакетами (бакетами)** для увеличения параллелизма
- Использует стратегию **owner-compute**: каждая вершина принадлежит одному процессу, который отвечает за её обработку

## Архитектура решения

### Распределение данных
- Граф распределяется по процессам с использованием **блочного разбиения вершин**
- Каждый процесс хранит только инцидентные рёбра для своих вершин
- Для обеспечения полноты графа реализован **обмен обратными рёбрами** между процессами

### Ключевые компоненты
1. **Генерация/загрузка графа**
   - Поддержка генерации случайных графов с заданной плотностью
   - Сохранение/загрузка графа в бинарные файлы для повторного использования
   - Автоматическое определение наличия предварительно сгенерированных графов

2. **Основной алгоритм Δ-шага**
   - Итеративная обработка бакетов
   - Разделение на фазы обработки лёгких и тяжёлых рёбер
   - Коллективные операции MPI для обновления расстояний

3. **Коммуникационная модель**
   - Использование `MPI_Alltoallv` для эффективного обмена запросами релаксации
   - Минимизация объёма передаваемых данных через локальную агрегацию
   - Измерение времени коммуникаций для анализа производительности

## Функциональные возможности
- Поддержка графов различных размеров (от 100 до 10000 вершин)
- Настраиваемая плотность рёбер
- Конфигурируемый параметр Δ
- Вывод расстояний до вершин на корневом процессе
- Замер времени выполнения различных этапов
